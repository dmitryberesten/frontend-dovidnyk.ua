Відео-уроки:

HTML & CSS

HTML & CSS

Мова розмітки гіпертексту. Готує певним чином деякий документ для того, щоб абияка програма могла його використовувати.

CSS 3

CSS 3

Каскадні таблиці стилів за допомогою яких описують зовнішній вигляд документів. Дозволяють творити неймовірні речі.

GIT

GIT

Розподілена система контролю версій, яка дозволяє відстежувати історію розробки і спільно працювати над складними проектами з будь-якої точки світу.

GitHub

GitHub

Є одним із безлічі сервісів на основі Git. Для легшого розуміння можна уявити багато розробників, де ті переглядають коди один одного, допомагають в розробці.

Sass

Sass

Це препроцесор, Він надає додатковий набір функцій, що робить процес написання стилів більш простим та швидким. Писати менше коду, зберегти свій час.

Bootstrap

Bootstrap

Фреймворк в якому є готові стилі і скрипти, для застосування яких вам достатньо всього лише прописати необхідні стильові класи і атрибути html-елементів.

JavaScript

JavaScript

Мова програмування яка допомагає зробити сторінки сайту більш інтерактивними, обробляє дії користувачів сайту.

Angular

Angular

Фреймворк JavaScript для розробки складних та високонавантажених сайтів та web-систем. Великі фрагменти готового коду з яких простіше зібрати готове рішення.

React

React

JavaScript-бібліотека. Можна порівняти компоненти React з пиріжками з начинкою, тільки зі світу веб-технологій. Вони можуть містити все, що ви тільки забажаєте.

Node.js

Node.js

Програмна платформа, що робить JavaScript мовою загального призначення. Вона вміє зв’язуватися з зовнішніми бібліотеками чи виконувати роль веб-сервера.
