Мінімальні знання JavaScript для роботи з React. Частина 1.

В даному міні курсі з тьох відео я розкажу та покажу що необхідно знати по JavaScript для роботи з бібліотекою React.Не всі хто використовує або змушений використовувати React добре знають JS і знають як з ним працювати. Я покажу той мінімум, якого буде достатньо для початку. Не потрібно бути експертом в JavaScript, щоб розпочати вивчення React або внесення якихось невеликих змін в уже присутній код.
